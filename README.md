# 🛒 React Product Cart App

A modern and responsive **Product Cart Application** built using **React, Tailwind CSS, DaisyUI, and ES6**. This project focuses on clean UI, smooth user experience, and interactive functionality like adding/removing products and checkout flow.

---

## 🚀 Tech Stack

* ⚛️ React (Component-based UI)
* 🎨 Tailwind CSS (Utility-first styling)
* 🌼 DaisyUI (Pre-built UI components)
* 📜 ES6 (Modern JavaScript features)

---

## ✨ Key Features

### 🧩 1. Dynamic Product Section

* Products are loaded dynamically from a **fake JSON dataset**
* Each product is displayed in a **card layout**
* Clean and responsive UI using Tailwind + DaisyUI

---

### ➕ 2. Add to Cart Functionality

* Click **"Add to Cart"** button to add products
* Instant feedback using **modern toast alerts**
* Cart updates dynamically

---

### ❌ 3. Remove Item from Cart

* Each cart item has a **Remove button**
* Clicking it removes the specific item instantly
* Smooth and user-friendly interaction

---

### 🛍️ 4. Proceed to Checkout

* Clicking **"Proceed to Checkout"**:

  * Clears all cart items
  * Shows alert: **"Cart is empty"**
* Simulates real-world checkout behavior

---

### 💰 5. Pricing Section

* Displays total price of all selected items
* Updates automatically when items are added/removed

---

### 🎯 6. Banner Section

* Attractive hero/banner section
* Highlights the purpose of the app
* Improves user engagement

---

## 🔥 How It Works

1. Load products from JSON
2. Display them as cards
3. Click **Add to Cart**
4. Items appear in cart section
5. Remove items individually if needed
6. Click **Checkout → Cart resets**

## 📦 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/pervezmia/digital-tools-platform.git

# Go to project folder
cd product-cart-app

# Install dependencies
npm install

# Run the app
npm run dev
```
